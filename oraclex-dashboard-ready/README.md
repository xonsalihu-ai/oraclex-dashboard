# OracleX Dashboard

Real-time trading intelligence dashboard powered by AI analysis and market state detection.

## Features

- **Market State Analysis** - Understand trend, volatility, and structure
- **Bias Stability Tracking** - See how long directional bias has been active
- **Confluence Breakdown** - Know what factors are driving the analysis
- **Context History** - Review 90 minutes of market evolution
- **Educational Insights** - Plain language interpretation of market conditions
- **Session Intelligence** - Automatic detection of Asia, London, and NY sessions

## Technology

- React 18
- Vite
- Tailwind CSS
- Lucide React icons
- Connected to Railway relay API

## Live

Deployed on Netlify with automatic updates from GitHub.

## About

OracleX is a context-first market intelligence tool. It clarifies market conditions without making predictions or trade recommendations. Built for traders who want to understand the market they're in, not the one they hope for.

"A second brain that checks your thinking."
